IdPathDetailsRequirements
qty-3CountIf a code for the unit is present, the system SHALL also be present
: code.empty() or system.exists()
cnt-3CountThere SHALL be a code with a value of "1" if there is a value and it SHALL be an expression of length. If system is present, it SHALL be UCUM. If present, the value SHALL a whole number.
: (code or value.empty()) and (system.empty() or system = %ucum) and (code.empty() or code = '1') and (value.empty() or value.toString().contains('.').not())
