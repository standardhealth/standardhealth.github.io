SHR EffectiveTimePeriod Profile

 

The official URL for this profile is: 

http://standardhealthrecord.org/fhir/StructureDefinition/shr-core-EffectiveTimePeriod

The date and time span for which something is active, valid, or in force.

This profile builds on Period.
