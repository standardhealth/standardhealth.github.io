HomeTelephoneNumber

A telephone number associated with a person's residence.
Based On: TelephoneNumber A sequence of decimal digits (0-9) that is used for identifying a destination telephone line or other device in a telephone network.
Value: TelecomNumberOrAddress A user name or other identifier on a telecommunication network, such as a telephone number (including country code and extension, if necessary), email address, or SkypeID.
TelecomType 1 The type or channel of the communication, such as email or telephone. If an messaging app, such as Skype, specify other.
        with code from http://hl7.org/fhir/ValueSet/contact-point-system
TelecomTypeOther optional Specification of the type of telecom channel, if the TelecomType is other.
TelecomQualifier optional Provides more information about the type and uses of the telecom address.
        with code is home
EffectiveTimePeriod optional The date and time span for which something is active, valid, or in force.
